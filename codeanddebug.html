Coding & Debugging


Description:

This event aims to measure your analytical, technical and programming skills set to solve and debug the code and also test your ability to work as a team. The participants who are interested in the hunt of coding excellency are heartly welcome.


General Rules & Regulations:

  • A team must have two members.
  • Based on performance teams will progress to the nextBased on performance teams will progress to the next.
  • Use of mobile phones or any electronic gadgets are not allowed.
  • The decision of the judges will be final and binding.

Round 1:

  • Part 1: Crack the password

    • It is an online round consisting of one question for every team.
    • The team member has to track the output and code the given problem.
    • After successful submission the team can proceed to part 2.
    • Duration 15 mins.

  • Part 2: Swapping logic

    • One coding question will be provided to each of the team member.
    • Each member will have to solve his problem as much as he can and after 15 mins both the team members will be swapped and they have to complete their remaining partner’s program.
    • Duration 35 mins.

Round 2: FINAL ROUND

  • Debug a given code and optimize it.
  • Given a program code with errors which must be debugged.
  • Complete the code and optimize it as much as you can
  • Duration 45 mins

NOTE:

  • If a member is not able to crack the password within the provided time, he/she will be given the password for login but score will be deducted.
  • Optimization shall be done according to given complexity.
